# Лабораторная 11

## Задание 1
Создать и настроить виртуальное окружение, а также репозиторий:
1. Все пакеты устанавливаются в виртуальную среду.
2. Репозиторий **не содержит** виртуальную среду.
3. Репозиторий содержит файл с зависимостями (`requirements.txt`).

## Задание 2
Создать программу, которая собирает (парсит) информацию о странах со следующего сайта:

**Источник данных:**
[Countries of the World: A Simple Example | Scrape This Site](https://www.scrapethissite.com/pages/simple/)

Программа должна выводить данные на экран в формате:
Country: Andorra; Capital: Andorra la Vella;
Country: Belarus; Capital: Minsk;
Country: Guadeloupe; Capital: Basse-Terre;

---

## Задание 3
Собранная информация сохраняется программой в файл **`data.json`**.

---

## Задание 4
На основе данных из файла **`data.json`** программа генерирует HTML-страницу (**`index.html`**) с красивым отображением информации:

### Требования к странице:
1. Страница имеет заголовок.
2. Страница имеет красивый фон (цвет, картинка или градиент).
3. Все собранные данные отображаются в таблице.
4. Таблица имеет оформление (границы, фон заголовков и т.д.).
5. Страница содержит ссылку на оригинальный источник данных.
